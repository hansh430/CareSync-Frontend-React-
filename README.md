# 💊 CareSync Frontend

A responsive React application for the CareSync Online Medicine Ordering System.

---

# 🚀 Live Demo

http://caresyncmed.runasp.net/my-orders

---

## Test the Application

Anyone can test the application by creating their own account.

### Customer Flow

1. Open the Live Demo
2. Click **Register**
3. Create a new account
4. Login
5. Browse medicines
6. Add medicines to cart
7. Place an order
8. View order history

No administrator account is required for testing customer features.

---

## Features

### Public

- Landing Page
- User Registration
- User Login

### Customer

- Browse Medicines
- Search Medicines
- Pagination
- Shopping Cart
- Wallet Balance
- Order History
- Order Details

### Administrator

- Dashboard
- Medicine Management
- Customer Management
- Wallet Management
- Order Management

- Vite
- Axios
- React Router
- Bootstrap 5
- Bootstrap Icons
- React Toastify

---

## Environment Variables

Create a `.env`

```env
VITE_API_URL=https://caresyncbackend.runasp.net/api

VITE_SERVER_URL=https://caresyncbackend.runasp.net
```

## Installation

```bash
git clone https://github.com/yourusername/CareSync-Frontend.git

cd CareSync-Frontend

npm install

npm run dev
```
